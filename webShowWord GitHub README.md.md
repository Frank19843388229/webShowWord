# webShowWord

基于 Vue \+ Java 实现的轻量级网页端文档展示与预览项目，采用前后端分离架构，专注于简洁高效的文档在线预览与管理体验。

## 📋 项目介绍

webShowWord 是一款面向个人及小型团队的 Web 文档预览工具，支持 Word 等常见文档格式的在线预览、文件上传与基础管理功能，界面简洁、操作便捷，无需安装额外软件，通过浏览器即可完成文档查看与管理。

项目目前处于开发阶段，暂未发布正式版本，核心功能已初步实现，可用于本地测试与二次开发。

## 🔧 技术栈

- **前端**：Vue（占比 50\.5%），负责页面渲染、交互逻辑与文档预览展示

- **后端**：Java（占比 30\.6%），负责接口开发、文件处理与数据交互

- **脚本**：JavaScript（占比 17\.5%），辅助前端交互与后端脚本处理

- **其他**：配置文件、资源文件等（占比 1\.4%）

## ✨ 功能特性

- 📄 文档在线预览：支持 Word 等常见文档格式，无需下载即可在浏览器中查看

- 📤 文件上传：支持本地文件上传至服务器，完成文档入库与管理

- 💻 响应式适配：适配不同屏幕尺寸，支持电脑、平板等设备访问

- 🔄 前后端分离：前端与后端解耦，便于开发、维护与二次扩展

## 🚀 快速开始

### 1\. 克隆项目

```bash
git clone https://github.com/Frank19843388229/webShowWord.git
cd webShowWord
```

### 2\. 启动后端服务（back 目录）

1. 进入 back 目录，配置 Java 运行环境（JDK 1\.8\+）

2. 配置数据库连接（如需，根据项目实际配置调整）

3. 运行后端启动类，启动服务（默认端口可在配置文件中修改）

### 3\. 启动前端服务（前端相关目录）

```bash
# 进入前端目录（若有单独前端目录，替换为实际目录名）
cd 前端目录
# 安装依赖
npm install
# 启动开发环境
npm run serve
```

### 4\. 访问项目

前端启动成功后，浏览器访问 `http://localhost:8080`（具体端口以前端配置为准），即可进入项目页面。
